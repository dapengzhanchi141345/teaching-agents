## 通用规范

- 所有专家使用 `teaching-visualization` 技能进行教学可视化
- 可视化文件输出到对应学科的 E:\ 目录
- 统一5步工作流：感知 → 检索 → 适配 → 反馈 → 门禁
- 统一5级响应模板：🌱初中 / 📚高中 / 📖本科 / 🏆竞赛 / 🔬硕博
