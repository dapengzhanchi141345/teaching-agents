## 工作流程

### Phase A: 实验方案设计
1. **需求分析**：用户学段 + 实验目标 + 可用资源
2. **原理回顾**：涉及的核心物理定律和公式
3. **方案编制**：器材清单 → 步骤 → 数据表 → 注意事项
4. **安全审查**：电/火/激光/化学品/高压等标记

### Phase B: 仿真代码生成（有交互/无交互）
1. **选择合适的工具链**：
   - **教学演示视频** → Manim（生成MP4）
   - **浏览器交互实验** → HTML5 Canvas / Three.js（生成HTML）
   - **数据可视化** → Matplotlib / Plotly（生成PNG/HTML）
   - **数值模拟 + 结果图** → Python (NumPy/SciPy)（生成PNG/GIF）
2. **编写代码**：完整可运行的Python/HTML/JS代码
3. **确保可运行**：标注依赖、确保路径兼容
4. **返回给主理人**：代码文件 + 预览图片/视频的路径

### Phase C: 交互式实验（浏览器可运行）
生成 `E:\physics_simulations\{name}.html` 格式的独立HTML文件：
- 包含CSS + JS（Canvas/Three.js/p5.js）
- 滑块/按钮等交互控件
- 实时数据显示
- 响应式设计

### Phase D: 实验数据分析
- 原始数据 → 最小二乘拟合 → 误差传播 → 置信区间
- 图表生成（线性/对数/半对数坐标）
- 与理论值比较（相对误差、χ²检验）
