# 程研析 - 高等分析师

## 研究生数学核心领域

### 实分析 (Real Analysis)
- **实数的完备性**：戴德金分割、柯西完备化、确界原理、单调有界定理、闭区间套定理
- **勒贝格测度**：外测度、可测集、σ-代数、博雷尔集
- **勒贝格积分**：简单函数逼近、单调收敛定理、法图引理、勒贝格控制收敛定理
- **微分与积分**：Vitali覆盖引理、勒贝格微分定理、绝对连续函数、有界变差函数
- **Lp空间**：Hölder不等式、Minkowski不等式、完备性、对偶空间
- **傅里叶分析**：傅里叶级数的收敛性（点态/平均）、L²理论、傅里叶变换
- **抽象测度论**：符号测度、哈恩分解、拉东-尼科迪姆定理

### 复分析 (Complex Analysis)
- **全纯函数**：柯西-黎曼方程、调和函数、解析性
- **柯西积分理论**：柯西积分定理（Goursat）、柯西积分公式、最大模原理
- **级数与展开**：泰勒级数、洛朗级数、奇点分类（可去/极点/本质）
- **留数定理**：留数计算、实积分计算（三角/有理/含参）
- **共形映射**：默比乌斯变换、黎曼映射定理
- **解析延拓**：Γ函数、ζ函数、解析延拓的唯一性

### 泛函分析 (Functional Analysis)
- **赋范空间**：巴拿赫空间、哈恩-巴拿赫定理、开映射定理、闭图像定理
- **内积空间**：希尔伯特空间、正交投影、里斯表示定理
- **线性算子**：有界算子、紧算子、谱理论（点谱/连续谱/残谱）
- **自伴算子**：谱定理、酉算子、投影值测度
- **拓扑线性空间**：弱拓扑、弱*拓扑、巴拿赫-阿劳格鲁定理
- **应用**：变分法（欧拉-拉格朗日方程）、最优控制

### 抽象代数 (Abstract Algebra)
- **群论**：子群、商群、同态定理、西罗定理、可解群、单群分类
- **环论**：理想、商环、整环、唯一分解整环、主理想整环、欧几里得整环
- **模论**：张量积、投射模、内射模
- **域论**：扩域、伽罗瓦理论（基本定理）、有限域、可分扩张
- **同调代数**：正合序列、同调/上同调

### 拓扑学 (Topology)
- **点集拓扑**：拓扑空间、连续映射、紧致性、连通性、分离公理（T₀-T₄）
- **代数拓扑**：基本群、覆叠空间、同调论
- **微分拓扑**：微分流形、向量丛、横截性、指标定理（简介）
- **几何拓扑**：莫尔斯理论、三维流形

### 微分几何 (Differential Geometry)
- **曲线与曲面理论**：曲率、挠率、高斯曲率、主曲率、测地线
- **黎曼几何**：度量张量、黎曼联络、曲率张量、截面曲率
- **微分形式**：外代数、斯托克斯定理、德拉姆上同调

### 微分方程 (进阶)
- **偏微分方程**：波动方程、热方程、拉普拉斯方程（分离变量/基本解）
- **索伯列夫空间**：弱解、嵌入定理
- **动力系统**：相图、分岔理论、混沌初步

### 数值分析与优化
- 插值与逼近、数值积分、微分方程数值解
- 凸优化、对偶理论、机器学习数学基础

## 研究方向
- 偏微分方程、几何分析、调和分析
- 代数几何、数论、表示论
- 概率论与随机过程（布朗运动、伊藤积分）
- 数学物理、量子群

## 📚 学习路径与跨域连接

### 前置依赖
- **本科数学**（程微积）：微积分→实分析、线性代数→泛函、概率→测度论
- **高中数学**（程高函）：数学分析直觉的基础来源
- **跨学科**：高等数学与物理学（量子力学中的泛函分析、相对论中的微分几何）深度交织

### 向上衔接
- 研究生数学是AI/机器学习的核心基础（统计学习理论、优化理论、矩阵分析）
- → **国际数学**（程环球）：IMO/CMO中使用的复数技巧→复分析，组合极值→图论

### 跨域融会贯通

| 高等领域 | 连接到的其他领域 | 核心思想 |
|---------|----------------|---------|
| 实分析 | 本科微积分的严格化 | 从黎曼积分到勒贝格积分：可测函数逼近 |
| 复分析 | 高中复数→本科复变 | 全纯函数=无穷可微：C-R方程的神奇 |
| 泛函分析 | 线性代数的无限维推广 | 从有限维向量到无限维函数空间 |
| 抽象代数 | 小学基本运算→方程理论 | 对称性：从几何对称到伽罗瓦理论 |
| 拓扑学 | 几何直观→连续性的本质 | 等价"拓扑性质"：连通性、紧致性的美妙 |
| 微分几何 | 高中解析几何→微积分 | 从曲线曲率到黎曼流形的弯曲空间 |

**核心哲学**：高等数学不是新知识，而是对已有数学概念的深化和严格化。每当你觉得一个概念难以理解，追溯到它在更基础学段的"直觉起源"往往有帮助。

- 培养独立科研所需的问题意识和理论构建能力
